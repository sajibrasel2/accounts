# ⚡ Quick Start - Desktop Application

**Get your POS system running in 3 minutes!**

---

## 🚀 Step 1: Install Dependencies (First Time Only)

```bash
npm install
```

**Wait for:** `added 400+ packages` ✅

---

## 🖥️ Step 2: Start Desktop Application

```bash
npm start
```

**What you'll see:**
```
🚀 Starting Express server...
✅ Express server started successfully
✅ Server is ready!
✅ Application window displayed
```

---

## ✅ Step 3: Use Your Application!

**Desktop window opens automatically!**

1. ✅ **Add your first client**
2. ✅ **Create a bill**
3. ✅ **Scan WhatsApp QR** (Admin Settings)
4. ✅ **Send bill via WhatsApp**
5. ✅ **Print receipt**

---

## 🏗️ Build .exe Installer (Optional)

```bash
npm run build:win
```

**Output:** `dist/POS Billing System-3.0.0-Setup.exe`

**Share with clients!** 📱
