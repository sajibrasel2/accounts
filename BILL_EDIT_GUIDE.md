# 🔧 বিল সংশোধন গাইড (Bill Edit & Resend Guide)

## ✨ নতুন ফিচার (New Features)

এখন আপনি যেকোনো বিল সংশোধন করে আবার WhatsApp-এ পাঠাতে পারবেন!

### 🎯 কী কী করতে পারবেন:

1. ✅ **বিল সংশোধন (Edit Bill)** - ভুল বিল ঠিক করুন
2. ✅ **আবার পাঠান (Resend WhatsApp)** - সংশোধিত বিল আবার পাঠান
3. ✅ **সরাসরি প্রিন্ট (Direct Print)** - যেকোনো বিল প্রিন্ট করুন

---

## 📝 কিভাবে বিল সংশোধন করবেন

### ধাপ ১: Dashboard-এ যান

1. Browser-এ খুলুন: `http://localhost:3000`
2. **Dashboard** tab-এ ক্লিক করুন
3. **Recent Transactions** টেবিল দেখুন

### ধাপ ২: Edit বাটনে ক্লিক করুন

প্রতিটি বিলের সাথে তিনটা বাটন দেখবেন:
- **নীল Edit আইকন (🖊️)** - বিল সংশোধন
- **সবুজ WhatsApp আইকন (📱)** - আবার পাঠান  
- **বেগুনি Print আইকন (🖨️)** - প্রিন্ট করুন

যে বিল সংশোধন করতে চান তার **Edit আইকনে** ক্লিক করুন।

### ধাপ ৩: সংশোধন করুন

Edit Modal খুলবে যেখানে দেখবেন:

#### ক্লায়েন্ট তথ্য (পরিবর্তন করা যাবে না):
- ক্লায়েন্টের নাম
- পূর্বের বকেয়া (Previous Due)

#### যা পরিবর্তন করতে পারবেন:

1. **আইটেম পরিবর্তন:**
   - আইটেমের নাম বদলান
   - সংখ্যা (Quantity) বদলান
   - দাম (Rate) বদলান
   - নতুন আইটেম যোগ করুন (Add Item বাটন)
   - আইটেম ডিলিট করুন (🗑️ আইকন)

2. **জমা প্রদান পরিবর্তন:**
   - "জমা প্রদান (Paid Amount)" বক্সে নতুন টাকা লিখুন
   - স্বয়ংক্রিয় "নতুন মোট বকেয়া" হিসাব হবে

3. **নোট পরিবর্তন:**
   - Additional notes যোগ বা পরিবর্তন করুন

### ধাপ ৪: Save করুন

- **"সংশোধন সেভ করুন"** বাটনে ক্লিক করুন
- ✅ Success message দেখবেন: "বিল সংশোধন হয়েছে!"
- Dashboard-এ নতুন তথ্য দেখবেন

---

## 📱 কিভাবে আবার WhatsApp পাঠাবেন

### Option 1: Edit করার পর পাঠান

1. বিল Edit করুন (উপরের ধাপ অনুসরণ করুন)
2. Save করুন
3. Dashboard-এ যান
4. সেই বিলের **WhatsApp আইকন (সবুজ)** ক্লিক করুন
5. Confirm করুন: "আবার WhatsApp পাঠাতে চান?"
6. ✅ Message পাঠানো হবে

### Option 2: কোনো বিল আবার পাঠান (Edit ছাড়া)

যদি বিল সঠিক আছে কিন্তু ক্লায়েন্ট message পায়নি:

1. Dashboard → Recent Transactions
2. সেই বিলের **WhatsApp আইকন** ক্লিক করুন
3. Confirm করুন
4. ✅ আবার পাঠানো হবে

---

## 🖨️ যেকোনো বিল প্রিন্ট করুন

Dashboard থেকে সরাসরি:

1. যে বিল প্রিন্ট করতে চান
2. **Print আইকন (বেগুনি)** ক্লিক করুন
3. Print dialog খুলবে
4. Print to PDF বা printer-এ পাঠান

---

## ⚠️ গুরুত্বপূর্ণ তথ্য

### লেজার (Ledger) হিসাব স্বয়ংক্রিয়:

যখন আপনি বিল সংশোধন করেন:

1. **পুরাতন Total Due** ক্লায়েন্টের balance থেকে বাদ যায়
2. **নতুন Total Due** ক্লায়েন্টের balance-এ যোগ হয়
3. অন্যান্য বিলের উপর কোনো প্রভাব পড়ে না

### উদাহরণ:

**আগের বিল:**
- Subtotal: ৳100
- Paid: ৳50
- Current Due: ৳50
- Previous Due: ৳0
- **Total Due: ৳50**

**Client Balance ছিল: ৳50**

**এখন Edit করলাম:**
- Subtotal: ৳150 (আইটেম যোগ করলাম)
- Paid: ৳70 (বেশি জমা দিল)
- Current Due: ৳80
- Previous Due: ৳0 (একই)
- **Total Due: ৳80**

**নতুন Client Balance হবে:**
```
Old Balance - Old Total Due + New Total Due
= 50 - 50 + 80
= ৳80
```

✅ হিসাব সঠিকভাবে update হয়!

## 🔍 কোন বিল সংশোধন করা উচিত?

### সংশোধন করুন যদি:
- ❌ আইটেমের নাম ভুল
- ❌ সংখ্যা (Quantity) ভুল
- ❌ দাম (Rate) ভুল  
- ❌ জমা প্রদান (Paid Amount) ভুল
- ❌ ক্লায়েন্টকে ভুল বিল পাঠিয়ে ফেলেছেন

### সংশোধন করবেন না যদি:
- ✅ বিল সঠিক কিন্তু ক্লায়েন্ট message পায়নি
  - এক্ষেত্রে শুধু **Resend** করুন
- ✅ শুধু আবার প্রিন্ট করতে চান
  - এক্ষেত্রে শুধু **Print** বাটন চাপুন

## 🎬 সম্পূর্ণ প্রক্রিয়া (Complete Workflow)

### Scenario: ভুল বিল পাঠিয়ে ফেলেছেন

**সমস্যা:**  
রাসেল সাহেবকে ৳100 এর বিল পাঠালাম, কিন্তু আসলে ৳150 হওয়া উচিত ছিল।

**সমাধান:**

1. **Dashboard** → Recent Transactions
2. রাসেল সাহেবের বিল খুঁজুন
3. **Edit আইকন** ক্লিক করুন
4. Edit Modal-এ:
   - একটা নতুন আইটেম যোগ করুন (৳50)
   - অথবা বিদ্যমান আইটেম পরিবর্তন করুন
   - "নতুন মোট বকেয়া" চেক করুন: ৳150
5. **"সংশোধন সেভ করুন"** ক্লিক করুন
6. Dashboard-এ ফিরে আসুন
7. **WhatsApp আইকন** ক্লিক করুন
8. Confirm করুন: "আবার WhatsApp পাঠাতে চান?"
9. ✅ সঠিক বিল পাঠানো হয়েছে!

রাসেল সাহেবের WhatsApp-এ নতুন message যাবে সঠিক ৳150 বিল নিয়ে।

## ❓ সমস্যা সমাধান

### সমস্যা: Edit বাটন কাজ করছে না

**সমাধান:**
1. Browser refresh করুন (F5)
2. Server চালু আছে কিনা চেক করুন
3. Console খুলে error দেখুন (F12)

### সমস্যা: Edit করার পর balance ভুল দেখাচ্ছে

**সমাধান:**
1. Dashboard reload করুন
2. Clients tab-এ গিয়ে client এর balance চেক করুন
3. সাধারণত স্বয়ংক্রিয় ঠিক হয়ে যায়

### সমস্যা: WhatsApp Resend কাজ করছে না

**সমাধান:**
1. Admin Settings → WhatsApp Connection চেক করুন
2. "Connected" দেখালে তবেই resend করুন
3. Connected না থাকলে QR scan করুন
