## ❓ Troubleshooting:

### Problem: "Cannot GET /admin_panel.html"
**Solution:** Files এখন public folder এ আছে। Server restart করুন:
```bash
# Server stop করুন (Ctrl+C)
# আবার চালু করুন
npm start
```

### Problem: Login কাজ করছে না
